# Van Zone Docs

Van Zone 开源组织的文档站点，基于 VitePress 构建。

## 项目列表

- [VanBot / Van-keyword](https://github.com/Van-Zone/VanBot) - NoneBot 词库插件
- [Van-Keyword-For-Web-API](https://github.com/Van-Zone/Van-Keyword-For-Web-API) - 词库 HTTP API 服务
- [VanAPI](https://github.com/Van-Zone/VanAPI) - 二次元动漫图片 API

## 本地开发

本站点构建为纯静态 HTML，基于 VitePress 框架，源文件为手动编写的 HTML 页面。
