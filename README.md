# Van Zone Docs

Van Zone 旗下三个开源项目的文档站，用 VitePress 搭的。

https://van-zone.github.io/Docs/

## 项目

### 🤖 VanBot — 机器人词库插件

适配 NoneBot 的 Python 词库。专为 VanBot 设计，集成即用，支持关键词匹配、变量替换、多词库管理。内置丰富变量系统，可拓展多样化功能。

- 平台：QQ、Kook
- 语言：Python 3.10+
- 许可证：MIT
- 维护：[@ZiYiQuQ](https://github.com/ZiYiQuQ)
- 源码：[Van-Zone/VanBot](https://github.com/Van-Zone/VanBot)
- 原版：[ZiYiQuQ/VanBot](https://github.com/ZiYiQuQ/VanBot)

### 🌐 Van-Keyword-For-Web-API — 词库 HTTP 服务

基于 FastAPI 的词库管理服务，把 VanBot 词库能力封装成 RESTful API，附带 WebUI 可视化管理界面。支持多机器人、多用户、多群组，带权限控制和完整变量系统。

- 版本：v3.3.9
- 语言：Python (FastAPI) + HTML
- 许可证：MIT
- 维护：[TY-China](https://github.com/TY-China)
- 源码：[Van-Zone/Van-Keyword-For-Web-API](https://github.com/Van-Zone/Van-Keyword-For-Web-API)

### 🎨 VanAPI — 二次元动漫图片 API

动漫图片制作 API，提供图片生成、图像处理等服务接口。

- 语言：Python
- 维护：[@ZiYiQuQ](https://github.com/ZiYiQuQ)
- 源码：[Van-Zone/VanAPI](https://github.com/Van-Zone/VanAPI)
