VanBot / Van-keyword 词库 ​

适配 NoneBot 的 Python 词库

为 VanBot 量身打造，提供高效、可扩展的关键词支持，无缝集成对话交互、指令匹配等核心功能，助力 VanBot 快速实现智能回复。

✨ 项目简介 ​

本词库基于 Python 开发并适配 NoneBot 框架，专为 VanBot 提供稳定、灵活的关键词管理、对话匹配及语义解析能力。

无需额外适配开发，可直接集成至 VanBot 项目中，快速实现关键词触发、自动回复、指令匹配等核心场景；同时支持自定义词库导入与扩展，轻松满足 VanBot 个性化开发与场景化需求。

本词库内置丰富变量，开发者可结合这些变量灵活拓展，实现多种多样的个性化功能，极大提升 VanBot 的交互能力与使用场景。

VanBot 已适配 QQ、Kook 等主流社交平台，本词库可同步适配所有 VanBot 支持的平台，确保多平台使用体验一致。

📌 核心特性 ​

  • 🧩 无缝集成 — 完美对接 VanBot 架构，集成即生效，无需额外开发
  • 🐍 版本兼容 — 基于 Python 3.10+ 开发，兼容 NoneBot 主流版本
  • 🔧 灵活扩展 — 支持 .txt .json 格式词库导入，关键词增删改查灵活操作
  • 📊 丰富变量支持 — 内置大量变量，可结合变量拓展多样化功能，适配各类个性化需求
  • 📄 开源免费 — 基于 MIT 许可证，可自由用于个人及商业项目，支持二次开发
  • ⚖️ 规范引用 — 使用本项目需标注来源，尊重开源成果与原作者权益
  • 🌐 多平台支持 — 同步适配 VanBot 已支持的 QQ、Kook 平台，适配无差异

🛠️ 快速集成 ​

  1. 克隆 VanBot 项目至本地：
    bash
    git clone https://github.com/ZiYiQuQ/VanBot.git
  2. 进入项目目录，安装对应依赖（需 Python 3.10+）
  3. 在 VanBot 配置文件中启用本词库
  4. 启动 VanBot，词库自动加载生效（QQ、Kook 平台同步可用）

详细集成文档、词库使用示例、变量用法及配置教程，请参考 Wiki（持续更新）

📖 词库结构 ​

  • 核心词库 — 内置 VanBot 基础指令、常用对话关键词，适配 QQ、Kook 平台对话场景
  • 扩展词库 — 预留独立自定义目录，支持外部词库导入，适配多样化使用需求
  • 变量模块 — 内置丰富变量，支持变量调用与组合，助力功能拓展
  • 配置文件 — 可灵活配置词库匹配规则、触发阈值等参数，按需调整功能效果

🔤 变量系统 ​

基础变量 ​

变量说明
[qq]用户 QQ 号
[name]用户昵称
[群号]群组 ID
[词条id]词条 ID
[词汇量]词库词条数
[当前词库]当前使用的词库名称
[n.1] ~ [n.5]匹配变量占位符

匹配模式 ​

  • 精确匹配 — 关键词必须完全匹配
  • 模糊匹配 — 消息包含关键词即可触发
  • 管理员专用 — 仅管理员可访问的特定词条
