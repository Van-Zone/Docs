关于 Van Zone ​

Van Zone 是一个开源组织，专注于开发聊天机器人和二次元动漫相关的工具项目。

组织项目 ​

  • VanBot / Van-keyword — 基于 Python 的 NoneBot 词库插件，支持 QQ、Kook 等平台
  • Van-Keyword-For-Web-API — 基于 FastAPI 的词库 HTTP 服务与 WebUI 管理界面
  • VanAPI — 二次元动漫图片制作 API

生态伙伴 ​

  • Van-lexicon — 多功能词库（由 ZiYiQuQ 维护）

社区 ​
