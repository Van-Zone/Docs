关于 Van Zone

专注于开源机器人词库、API 服务与二次元工具的开源组织

组织简介

Van Zone 是一个活跃的开源组织，致力于构建聊天机器人生态的完整工具链。旗下项目覆盖词库引擎、HTTP API 服务、二次元图像处理等多个领域。

旗下项目

项目描述链接
VanBot / Van-keyword基于 NoneBot 的 Python 词库插件，支持 QQ、Kook 多平台GitHub
Van-Keyword-For-Web-API基于 FastAPI 的词库 HTTP 服务，带 WebUI 管理界面GitHub
VanAPI二次元动漫图片制作 APIGitHub
