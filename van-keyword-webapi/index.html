Van-Keyword-For-Web-API

基于 FastAPI 的词库 HTTP 服务 — 带 WebUI 可视化管理界面

📦 项目地址： github.com/Van-Zone/Van-Keyword-For-Web-API  |  版本：v3.3.9

📋 项目概述

Van-Keyword-For-Web-API 是一个基于 FastAPI 的 HTTP 服务，将 VanBot 词库能力封装为标准的 RESTful API。支持关键词查询、词库管理、消息解码等功能。

内置功能完整的 WebUI 管理界面，支持可视化操作，无需命令行即可管理词库、测试功能、查看服务器状态。

✨ 核心特性

⚡ API 功能

  • 关键词查询 — 精确匹配与模糊匹配两种模式
  • 词库管理 — 添加、删除、修改关键词及其回复
  • 多词库支持 — 用户、群组、公共词库多层级管理
  • 权限控制 — Bearer Token 双重验证 + 管理员权限
  • 消息解码 — 完整变量替换、多媒体消息处理
  • CQ 码兼容 — 自动转换常见 CQ 码格式

🧩 高级变量系统

功能语法示例
时间变量(Y) (M) (D) (h) (m) (s)(Y)年(M)月(D)日 → 2026年7月7日
数学运算(+表达式)(+1+2) → 3
随机数(最小值-最大值)(1-100) → 42
冷却时间(秒数~)(60~) → 60秒冷却
条件判断{a>b} {a=b}数值比较和字符串判断
分句发送(-秒数-)多句消息分开发送

🌐 WebUI 功能

  • 服务器状态 — 实时监控运行状态、连接测试
  • 关键词查询 — 交互式查询界面
  • 消息解码 — 变量替换测试面板
  • 词库管理 — 可视化添加/删除词条
  • 工具集 — 消息转码、JSON 格式化、管理员管理
  • 示例代码 — curl 和 JavaScript 调用示例

🛠️ 技术栈

  • Python 3.8+
  • FastAPI + Uvicorn（异步 Web 框架）
  • HTML/CSS/JavaScript（WebUI 前端）
  • JSON 文件存储（轻量无数据库依赖）

🚀 快速部署

git clone https://github.com/Van-Zone/Van-Keyword-For-Web-API.git
cd Van-Keyword-For-Web-API
pip install fastapi uvicorn httpx pydantic
python Van_keyword_WebAPI.py

🔌 API 接口概览

端点方法说明
/api/v1/keywordPOST关键词查询、解码、管理(增删改查)
/api/v1/examplesGET获取 API 使用示例
/statusGET获取服务器状态
/webuiGETWebUI 管理界面
/docsGETSwagger API 文档

📁 数据存储结构

Van_keyword_data/
├── {bot_id}/                    # 机器人独立目录
│   ├── lexicon/                 # 词库文件
│   │   ├── M_{user_id}.json    # 用户个人词库
│   │   └── common.json         # 公共词库
│   ├── config/                  # 配置文件
│   ├── expand/                  # 扩展变量文件
│   └── cooling/                 # 冷却相关文件
├── qq.txt                       # 管理员列表
└── Van_keyword_token.txt       # API Token

🔐 安全机制

  • 自动生成 16 位随机 Token
  • Bearer Token 双重验证机制
  • 管理员权限控制
  • Token 加密存储
