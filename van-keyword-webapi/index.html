Van-Keyword-For-Web-API ​

基于 FastAPI 的词库 HTTP 服务，将 VanBot 词库能力封装为 RESTful API，附带 WebUI 可视化管理界面。

项目概述 ​

VanBot 词库 API 是一个基于 FastAPI 的 HTTP 服务，提供关键词词库的查询、管理和存储功能。该系统主要用于聊天机器人场景，支持多机器人、多用户、多群组的词库管理，并提供了完善的 API 接口供外部调用。

新特性：集成了功能完整的 WebUI 管理界面，支持可视化操作，无需命令行即可管理词库、测试功能、查看状态。

📌 主要特性 ​

核心功能 ​

  • 关键词查询 — 支持精确匹配和模糊匹配两种模式
  • 词库管理 — 添加、删除、修改关键词及其回复
  • 多词库支持 — 用户、群组、公共词库多层级的词库管理
  • 权限控制 — 管理员权限机制和词条访问控制
  • 变量支持 — 支持 [n.数字] 格式的变量替换
  • CQ码兼容 — 自动转换常见 CQ 码格式

变量系统 ​

变量说明示例
[qq]用户 QQ 号123456
[name]用户昵称小明
[群号]群组 ID987654
(Y)年份2024
(M)月份12
(D)日期25
(h)小时14
(m)分钟30
(s)秒数45
(+1+2)数学运算3
(1-100)随机数42
(60~)冷却时间60 秒冷却
{a>b}条件判断true/false

WebUI 功能 ​

  • 可视化操作界面 — 浏览器即可管理所有功能
  • 实时状态监控 — 服务器状态、连接测试、功能验证
  • 交互式测试 — 关键词查询、消息解码、词库操作
  • 词库可视化 — 词条列表、搜索、统计信息
  • 工具集成 — 消息转码、JSON 格式化、管理员管理

技术特性 ​

  • RESTful API — 基于 FastAPI 的标准 HTTP 接口
  • Token 验证 — Bearer Token 双重验证机制
  • 异步处理 — 全异步架构，高性能处理
  • 日志系统 — 完整的操作日志记录
  • 文件存储 — 本地 JSON 文件存储词库数据

🏗️ 系统架构 ​

数据存储结构 ​

Van_keyword_data/
├── {bot_id}/                    # 机器人独立目录
│   ├── lexicon/                 # 词库文件
│   │   ├── M_{user_id}.json    # 用户个人词库
│   │   └── common.json         # 公共词库
│   ├── config/                  # 配置文件
│   ├── expand/                  # 扩展变量文件
│   └── cooling/                 # 冷却相关文件
├── qq.txt                       # 管理员列表
└── Van_keyword_token.txt       # API Token 备份

🔌 API 接口 ​

查询关键词 ​

POST /api/v1/keyword

json
{
  "action": "query",
  "botid": 123456,
  "userid": 789012,
  "groupid": 112233,
  "msg": "查询内容",
  "mode": 0,
  "token": "***"
}

消息解码 ​

POST /api/v1/keyword

json
{
  "action": "decode",
  "botid": 123456,
  "userid": 789012,
  "text": "你好！现在是(Y)年(M)月(D)日",
  "event_data": { "user_id": 789012, "group_id": 987654, "self_id": 123456 },
  "token": "***"
}

其他接口 ​

Action说明
add添加词条
remove删除词条
add_r为词条添加回复选项
remove_r删除词条回复选项
search搜索关键词
list列出所有词条
count统计词条数量
test测试接口连通性
transcodeCQ 码格式转换
admin_manage管理员管理
get_config获取系统配置

其他端点：

  • GET /api/v1/examples — 获取 API 使用示例
  • GET /status — 服务器状态信息
  • GET / — API 根目录信息

🚀 安装部署 ​

环境要求 ​

  • Python 3.8+
  • pip 包管理器

安装依赖 ​

bash
pip install fastapi==0.104.1 uvicorn==0.24.0 httpx==0.25.1 pydantic==2.5.0

启动服务 ​

bash
python Van_keyword_WebAPI.py

或使用 uvicorn：

bash
uvicorn Van_keyword_WebAPI:api_app --host 0.0.0.0 --port 8889 --reload

访问 WebUI ​

启动后，在浏览器中打开：http://localhost:8889/webui

⚙️ 配置选项 ​

python
API_HOST = "0.0.0.0"       # 监听地址
API_PORT = 8889            # 监听端口
MISTAKE_TURN_TYPE = False  # 是否自动转换中文符号
ENABLE_ADVANCED_FEATURES = True  # 启用高级功能
LOG_LEVEL = "INFO"         # 日志级别
